BCIT GIS Practicum - Andrew Dollak

Eagle Mapping 2025

✨ Aerial Imagery Collection and Processing ✨

Abstract

This practicum with Eagle Mapping focused on aerial imagery collection and processing techniques for topographic mapping and GIS analysis. Working with industry-standard equipment and software, I participated in flight planning, image acquisition, and data processing workflows to produce high-quality orthoimagery and digital elevation models. The project addressed challenges in data management, quality control, and integration with existing GIS systems while adhering to industry specifications for spatial accuracy.

Eagle Mapping Flight Planning:

The aerial survey mission covered Germansen Landing and Manson Creek in northern British Columbia. SnapPlan software was used to design optimal flight paths based on terrain and coverage requirements.

Flight Parameters:
  • Reference ground height: 1180m (3871 ft)
  • Flying height: 2880m (9449 ft)
  • Altitude range: 1647-1954m above terrain
  • Optimized for consistent 10cm GSD
  • Enhanced overlap (60% forward, 30% side)
  • Camera: PhaseOne iXM 100MP
Planning Considerations:

Flight lines were strategically oriented to minimize sun glare and shadows. Terrain-adaptive altitude adjustments were implemented to maintain consistent ground sampling distance across elevation changes. Google Earth was used for pre-flight validation of coverage and for visualizing the final swath patterns.

✨ SnapPlan Software ✨

This classic interface provides specialized tools for flight planning including strip creation, coverage analysis, terrain compensation, altitude optimization, and overlap calculation. Field Work & Equipment Setup
Ground Control Setup:

Field work involved establishing 12 ground control points (GCPs) across the project area. Each GCP was surveyed using RTK GNSS methods to obtain precise coordinates.

Field Equipment:
  • Trimble R10 RTK GNSS receivers
  • Field tablets with data collection software
  • Visible ground targets (1m x 1m)
  • Radio communication equipment
GNSS equipment and ground targets setup
Accuracy Requirements:
Ground control points were established with horizontal accuracy of ±2cm and vertical accuracy of ±3cm to ensure final product quality met industry specifications.
Data Analysis Techniques
Point cloud classification and analysis process
LiDAR Data Classification:
  • Ground point identification using progressive densification
  • Vegetation height stratification (low, medium, high)
  • Building and structure identification
  • Water feature extraction using intensity values
Quality Control Methods:
  • Statistical outlier removal
  • Cross-validation with independent checkpoints
  • Profile view inspection for classification errors
  • Density analysis to identify data gaps
Classification Accuracy:
Automated classification achieved 92% accuracy for ground points, with manual refinement improving to 97% final accuracy.

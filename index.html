Theory
How LiDAR Works:

LiDAR (Light Detection and Ranging) is a remote sensing method that uses light in the form of pulsed lasers to measure distances to the Earth's surface.

Key Components:
  • Laser: Emits pulses of light
  • Scanner: Directs laser pulses
  • Photodetector: Measures returned signals
  • GPS/IMU: Tracks position and orientation
Pulse Types:
Type Use
NIR Topographic
Green Bathymetric
Return Types:

A single laser pulse can have multiple returns:

  • First Return: Typically tree canopy
  • Intermediate Returns: Vegetation layers
  • Last Return: Often ground surface
Point Cloud Generation:

LiDAR creates detailed 3D representations through:

  • Time of Flight: Measures time between pulse emission and return
  • Coordinate Assignment: Combines with GPS/IMU data for XYZ position
  • Point Cloud Formation: Millions of points create 3D representation
LiDAR Terminology
LiDAR Terminology Glossary - Common terms used in LiDAR mapping
AGL
Above Ground Level - Height measurement from ground surface rather than sea level.
PPM
Points Per Meter - the number of points for each imaged meter, AKA resolution.
DSM
Digital Surface Model - Represents the earth's surface including all objects (buildings, vegetation).
DTM
Digital Terrain Model - Representation of bare ground surface without objects.
PRR
The diameter of the beam at the target's range.
Laser Footprint
Ground Sampling Distance - Distance between pixel centers on the ground.
IMU
Inertial Measurement Unit - Device that measures aircraft orientation and movement.
LAS/LAZ
Industry-standard file formats for LiDAR point clouds (LAZ is compressed).
Point Cloud
Collection of data points in 3D space, each with X, Y, Z coordinates.
Max Range
The maximum achievable distance at which a scanner can perform range measurements.
GNSS
Global Navigation Satellite System - Satellite navigation system with global coverage.
Swath
The lateral spread of LiDAR pulses over terrain. Swath width increases at higher AGLs.
Overlap
The point at which parallel swaths overlap, resulting in additional coverage.
Abstract

My practicum with Eagle Mapping focused on aerial imagery acquisition and processing workflows for high-precision topographic mapping. The practicum involved flight planning, aerial data collection, data processing, and point cloud classification. Working with industry-standard equipment and software, I participated in the complete LiDAR workflow from acquisition to final product delivery, addressing challenges in data management, quality control, and integration with existing GIS systems while adhering to industry specifications for spatial accuracy.

LiDAR Workflow Overview
✨ Key Components ✨

The product workflow at Eagle Mapping involves a series of integrated steps resulting in high-precision topographic data. Flight planning ensures optimal coverage and point density, while acquisition captures millions of data points per second using aerial LiDAR sensors. Processing involves trajectory correction and data alignment, followed by classification to identify different surface types. Final products include digital terrain models (DTMs), digital surface models (DSMs), and high resolution imagery used in various GIS applications.

Flight Planning
Drawing flight lines over Digital Elevation Models in SnapPlan
Creating a flight plan:
  • Lines are drawn according to required overlap
  • The plan is created over a DEM for terrain variations
  • Some areas require special plans. Push plans, for example, ensure that when the required point density is not being met, like when flying over valleys in mountaineous regions, triple coverage is achieved
LiDAR Acquisition
Acquisition Equipment:
  • Aircraft: Eagle Mapping owns and leases several aircraft throughout North America
  • Planes typically fly between 120-150kts
  • These planes are equipped with various RIEGL LiDAR scanners, such as the VQ-1560II-S pictured
  • Medium format cameras, such as the 150 megapixel PhaseOne iXM-RS150F, are used for ultra-high resolution imagery
LiDAR Operators:
  • Acquisition could not be achieved without the hard work of a LiDAR Operator
  • Their job is to ensure that data is being recorded properly and that specs are being met
  • If the equipment fails or the plane was flying too fast, low, or high, it's the operator's job to communicate with the pilot that a line needs to be reflown
  • Operators may fly several hours a day, for several days on end, far away from home
  • Eagle Mapping conducts operations across North America inculding Canada, the US, Mexico, Columbia, Guatemala and more

LiDAR Processing
Trajectory Processing:

A critical first step in LiDAR processing involves accurately determining the aircraft's position and orientation throughout the flight. This is achieved by processing GNSS and IMU data using Applanix POSPac MMS software, generating a Smoothed Best Estimate of Trajectory (SBET) file.

Flight trajectory over Vancouver
Resulting point cloud in RiProcess
Processing Workflow:
  1. Import GNSS/IMU data to POSPac
  2. Process data to generate SBET solution
  3. Import SBET and raw LiDAR data to RiProcess
  4. Apply system calibration parameters
  5. Georeference points to trajectory
  6. Generate point cloud with coordinate transform
Quality Control:
  • GNSS position accuracy: <5cm
  • IMU attitude accuracy: <0.005°
  • Point cloud vertical accuracy: ±15cm RMSE
  • Flight line overlap consistency: ±8cm
Point Cloud Generation:

After trajectory processing, the SBET file is combined with raw laser measurements in RiProcess to generate an accurately georeferenced point cloud. Each point is assigned precise XYZ coordinates based on the aircraft's position, scanner angle, and measured distance. The resulting point cloud is the foundation for all downstream LiDAR products.

Classification Process:

Point cloud classification follows a semi-automated approach using TerraSolid software. First, algorithmic methods identify ground points using a progressive densification method. Next, vegetation points are classified based on height above ground and return characteristics. Buildings are identified using planar surface detection. Manual review and refinement ensures classification accuracy exceeds 98% when compared to control samples.
